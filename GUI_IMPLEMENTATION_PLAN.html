Key User Journeys

Journey A: Understanding Character Knowledge

  1. Find event "AI Reveals Intent" on Timeline
  2. Click event for details panel
  3. See "Facts Created: AI is malicious"
  4. Click "Who Knows This?"
  5. View: Captain: No, Engineer: Yes
Result: Quick epistemic check without leaving timeline

Journey B: Tracking Character Arc

  1. Navigate to Characters, select "Captain"
  2. See Arc Card with progress bar
  3. View transformation: Lie ‚Üí Truth
  4. Progress shows "Midpoint phase"
  5. Click Power Drawer for details
Result: Arc status visible at a glance

Journey C: Finding Unfired Setups

  1. Navigate to Story Logic
  2. Click "Setups & Payoffs" tab
  3. See warning: setup unfired for 7 chapters
  4. Click for original plant event
  5. Plan payoff in climax
Result: Prevents abandoned plot threads

Journey D: Validating Consistency

  1. Validation screen ‚Üí "Run All Checks"
  2. View: 2 errors, 5 warnings
  3. Click error ‚Üí Jump to relevant view
  4. Power Drawer confirms issue
  5. Fix and re-validate
Result: Catches errors before publication
